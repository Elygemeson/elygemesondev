# Configuração do Ambiente

## Arquivo .env.local

Crie um arquivo `.env.local` na raiz do projeto com o seguinte conteúdo:

```
EMAIL_APP_PASSWORD=jkkp niqm yfls ngfr
EMAIL_FROM=elygemeson@gmail.com
EMAIL_TO=elygemeson@gmail.com
```

## Como criar o arquivo:

1. Na raiz do projeto, crie um novo arquivo chamado `.env.local`
2. Cole o conteúdo acima no arquivo
3. Salve o arquivo

**Importante:** O arquivo `.env.local` não deve ser commitado no Git (já está no .gitignore).
